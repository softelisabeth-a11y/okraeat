# React + Vite

🍽️ Restaurant Finder

Une application web permettant de découvrir des restaurants et les mets qu'ils proposent en fonction du type de repas : petit déjeuner, déjeuner et dîner.
Le projet permet également de consulter les prix minimum des repas, l’adresse des restaurants et leurs heures d’ouverture.

📌 Objectif du projet

L'objectif de cette application est de faciliter la recherche de restaurants en offrant aux utilisateurs une interface simple pour :

Consulter la liste des restaurants

Voir les menus proposés selon le moment de la journée

Connaître le prix minimum d’un repas

Trouver l’adresse et les horaires d’ouverture
